是的，Next.js 提供了官方脚手架工具 `create-next-app`，可以用一行命令快速生成一个 Next.js 应用作为开发基础[ref:1,6]。

## 创建命令

最简单的命令是：

```bash
npx create-next-app@latest
```

这个命令会启动交互式向导，让你配置项目名称和选项[ref:1,2]。

如果想直接指定项目名称并跳过所有提示（使用默认配置），可以使用：

```bash
npx create-next-app@latest my-app --yes
```

这样会自动创建一个包含基础页面（类似 Hello World 的欢迎页面）的 Next.js 应用，使用 TypeScript、ESLint、Tailwind CSS 和 App Router 等推荐默认配置[ref:1]。

## 是否需要安装 Node.js？

**是的，必须安装 Node.js**。Next.js 需要 Node.js 环境才能运行[ref:10,13,15]。目前 Next.js 要求：

- Node.js 版本 **≥ 18.17.0**（较新版本可能要求 ≥ 20.9.0）[ref:10,13]

你需要先安装符合版本要求的 Node.js，然后才能使用 `npx create-next-app` 命令来创建项目[ref:15]。

创建完成后，进入项目目录运行 `npm run dev` 即可启动开发服务器，访问 `http://localhost:3000` 就能看到生成的初始页面。
